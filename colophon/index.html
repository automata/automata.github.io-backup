This site is made by a mix of open source tools. It’s edited using Vim + tmux running either on a Ubuntu powered remote box I connect to through Mosh from a MacBook 12”, or from the laptop itself. The site is powered by Hexo, a static site generator made in Node.js and hosted in GitHub as Markdown files that I keep in the Git SCM. As theme it uses a modified version of Cactus Dark by Pieter Robberechts.

The art of the site are made generally by hand-drawing on Krita and vectorizing on Inkscape. I usually plug-in a Wacom tablet to make it easier to draw.
